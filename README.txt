ESG Passport v1.0.0
====================

Your sustainability data, always ready. Respond to ESG questionnaires in minutes.


QUICK START
-----------

1. Unzip this folder anywhere on your computer
2. Open the folder
3. Double-click "index.html" — it opens in your browser
4. That's it. You're running ESG Passport.

Your data is stored in your browser (localStorage). Nothing is sent anywhere.


WHAT YOU CAN DO FOR FREE
------------------------

- Track monthly ESG data (energy, water, waste, workforce)
- Auto-calculated emissions (Scope 1, 2, 3)
- Import data from CSV files
- Policy library
- VSME-aligned metrics
- Year-over-year trend tracking


UNLOCK THE RESPONSE ENGINE
---------------------------

With a license key (one-time purchase), you can also:

- Upload any ESG questionnaire (Excel, CSV, PDF, Word)
- Get draft answers from 200+ templates matched to your data
- Use pre-loaded templates (EcoVadis, CDP, CSRD/VSME)
- Export in multiple languages (EN, DE, FR, ES)

Purchase a license key: https://esgforsuppliers.lemonsqueezy.com
Then go to Settings in the app and enter your key.


AI ENHANCEMENT (OPTIONAL)
--------------------------

You can optionally enhance generated answers using your own AI API key:
- Go to Settings > AI Enhancement
- Enter your Anthropic (Claude) or OpenAI API key
- The app calls the API directly from your browser


DEMO DATA
----------

The included demo-data.csv file contains 12 months of sample ESG data.
Import it via the Data page > "Import CSV" to see the app in action.


UPDATING
---------

To update: download the latest version from https://esgforsuppliers.com/passport
and replace these files. Your data is stored in your browser, not in these files,
so it will still be there after updating.
